 نظام المطابقة الذكية (Matching Engine)
النظام يقارن تلقائياً بين الطلبات والعروض بناءً على:✅

	•	نوع العقار ✅
	•	الاستخدام ✅
	•	الموقع ✅
	•	المساحة ✅
	•	الميزانية / السعر ✅

ناتج المطابقة:
	•	قائمة تطابقات محتملة✅
	•	نسبة توافق (مثلاً 85٪)✅
	•	اسم وسيط العرض✅
	•	اسم وسيط الطلب✅



 الإشعارات والتنبيهات

إذا صار تطابق:
	•	إشعار داخل الداش بورد
	
	•	تنبيه للوسيطين:✅
	•	وسيط العرض ✅
	•	وسيط الطلب✅ 


	•	حالة الإشعار:✅
	•	جديد✅
	•	تم التواصل✅
	•	صفقة جارية✅
	•	مغلقة✅





النظام يدعم:
	•	إضافة مستخدم واحد أو 100 وسيط أو أكثر-✅
    ----------------------
	•	أنواع الصلاحيات:
	•	Manager (إدارة عروض وطلبات)

	•	Admin (كامل الصلاحيات) ✅
	•	Broker / وسيط (يدخل عروضه وطلباته فقط) ✅
	•	كل وسيط يشوف:✅
	•	عروضه ✅
	•	طلباته الخاصة ✅
